HTML Cheatsheet

Structure

This is the basic template or barebone structure of HTML.

Boilerplate

  1. HTML Boilerplate: This is the basic html boilerplate code 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. Heading: There are six headings available in HTML, H1 is the largest among all, and H6 is the smallest.

    <h1> tag

    <h1>Heading 1</hl>

  3. <h2> tag

    <h2>Heading 2</h2>

  4. <h3> tag

    <h3>Heading 3</h3>

  5. <h4> tag

    <h4>Heading 4</h4>

  6. <h5> tag

    <h5>Heading 5</h5>

  7. <h6> tag

    <h6>Heading 6</h6>
  8. Container: Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

    <div> tag

    div tag or division tag is used to make blocks or divisions in the document. 
    <div>This is div block</div>

  9. <span> tag

    span is a container for inline content. 
    <span>This is span block</span>

  10. <p> tag

    paragraph 
    <p>This is paragraph</p>

  11. <pre> tag

    pre tag represents pre-formatted text 
    <pre>Hello World</pre>

  12. <code> tag

    code tag is used to represent source codes 
    <code>
  import python
</code>

  13. Text Formatting

    Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

  14. <b> tag

    b tag represent bold letter 
    <b>I'm bold tag</b>

  15. <strong> tag

    <strong>I'm important text</strong>

  16. <i> tag

    <i>I'm italic text</i>

  17. <em> tag

    <em>Emphasized text</em>

  18. <sub> tag

    <sub>Subscript</sub>

  19. <sup> tag

    <sup>Superscript</sup>

  20. List tag

    Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for the clean document.

  21. <ol> tag

    Ordered list starts with <ol> tag and each list item starts with <li> tag 
    <ol>
  <li>Data 1</li>     
  <li>Data 2</li>     
  <li>Data 3</li>     
</ol>

  22. <ul> tag

    <ul>
  <li>Your Data</li>     
  <li>Your Data</li>    
</ul>

  23. Media tag

    Media is anything that is present in digital form such as image, video, audio, etc.

  24. <audio> tag

    It is used to embed sound content in the document. 
    <audio controls>
    <source src="cwa.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

  25. <img> tag

    It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">

  26. <video> tag

    It is used to embed or video in a webpage. 
    <video width="480" height="320" controls>
    <source src="cwa.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

  29. Link

    Links are clickable text that can redirect you to some other page.

  30. <video> tag

    It is used to embed or video in a webpage. 
    <a href="https://huntingcoder.me/">Visit huntingcoder.me!</a>

  31. Form

    Sample Form Form is used to collect user's input, generally user's data is sent to server for further processing. 
    <form action="/action.php" method="post">
    Name: <input name="name" type="text" /> <br />
    Age: <input max="90" min="1" name="age" step="1" type="number" value="18" /> <br />

    <input type="submit" value="Submit" />

  32. Semantic elements

    Semantic elements are those elements that are self describable, i.e., from their name itself, you can understand their meaning.

  33. <section> tag

    It defines a section in the document 
    <section>This is section</section>

  34. <article> tag

    It represents self-contained content 
    <article>Enter your data</article>

  35. <aside> tag

    It is used to place content in the sidebar 
    <aside>Your data</aside>

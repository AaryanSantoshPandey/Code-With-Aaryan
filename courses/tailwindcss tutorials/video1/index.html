So guys firstly what is tailind css? Ans: Tailwind css is the framework of css which help to style your website in html file by usingc class name.

and tailwindcss has this amazing ability to style your website without leaving html page.

Code as described/written in the video

  1. Code: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://cdn.tailwindcss.com"></script>
    <title>Tailwind CSS</title>
</head>
<body>
    <div>
      <nav>
        <ul class="flex bg-purple-900"> 
          <li class="p-2 text-white cursor-pointer">Home</li>
          <li class="p-2 text-white cursor-pointer">About</li>
          <li class="p-2 text-white cursor-pointer">Contact</li>
        </ul>
      </nav>
    </div>
</body>
</html>

    2. How to install tailwind css proper

  2. First run this command on your terminal 
    npm install -D tailwindcss postcss autoprefixer
  3. Second run this command on your terminal npm install vite
  4. third run this command on your terminal 
    npx tailwindcss init
  5. Than you have to add this code on your your tailwind.config.js file

    Why we are doing this because we have to add shortcut to write the classes in the tag that why we are adding this code.

    /** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["*"],
  theme: {
    extend: {},
  },
  plugins: [],
}
  6. Than you have to add this code on your your package.json file

    you have to add the scripts code on your package.json file to run the website on the localhost

    {
"scripts": {
    "start": "vite"
},
"devDependencies": {
    "autoprefixer": "^10.4.13",
    "postcss": "^8.4.21",
    "tailwindcss": "^3.2.6"
},
"dependencies": {
    "vite": "^4.1.1"
}
}

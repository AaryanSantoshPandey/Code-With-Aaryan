CSS Cheatsheet

Font

There are many properties related to the font, so such as the face, weight, style, etc. These properties allow you to change the style or complete Look to text.

Font Family

 
  1. font-family: 'Roboto', sans-serif;

Font Style

 
  1. font-style: italic;

Font-Variant

 
  1. font-variant: small-caps;

Font-Weight 

  1. font-weight: bold;

Font-Size

 
  1. font-size: larger;

Font

 
  1. font: style variant weight size family;

Text

Text properties allow one to manipulate alignment, spacing, decoration, indentation, etc., in the document.

Text-Align

 
  1. text-align: justify;

Letter-Spacing

 
  1. letter-spacing: .15em;

Text-Decoration

 
  1. text-decoration: underline;

Word-Spacing

 
  1. font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;

Text-Transform

 
  1. text-transform: uppercase;

Text-Indent

 
  1. text-indent: 0.5cm;

Line-Height

 
  1. line-height: normal;
